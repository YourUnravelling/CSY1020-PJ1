A generalised SQLite editor, using the contents of the SQLite file and config.py

AS OF THE CURRENT BUILD, lots of features are partially implemented and so have been disabled.

Currently over 2000 lines of code


# Modules / folders

## config
Usecase-dependant files which define behaviour and database

## resources
Usecase-dependant files which are non-source code

## gui
All tkinter and other graphical elements

## core-resources
All images and other resources that are internal and will be used regardless of the database

## core
Non-graphical elements
